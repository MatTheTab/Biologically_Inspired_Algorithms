Solves the QAP problem with a simple anti-\/heuristics (tries to find a very poor solution quickly). 

The antiheuristic works as follows\+:
\begin{DoxyEnumerate}
\item Compute the row sums for both matrices\+:
\begin{DoxyItemize}
\item {\ttfamily matrixA} (flow matrix)\+: Represents the interaction intensity between facilities.
\item {\ttfamily matrixB} (distance matrix)\+: Represents the physical distances between locations.
\end{DoxyItemize}
\item Rank the facilities based on descending order of interaction intensity (higher flow first).
\item Rank the locations based on descending order of distance sum (farther locations first).
\item Assign the facility with the highest flow sum to the location with the highest distance sum, the second-\/highest flow facility to the second-\/farthest distance location, and so on.
\end{DoxyEnumerate}

This method aims to decrease the risk of multiplying high values in matrix A and matrix B by one another, achieving relatively poor solution quickly.

The function follows these steps\+:
\begin{DoxyEnumerate}
\item Generate all possible 2-\/opt moves (swaps of two positions).
\item Evaluate each move by computing its change on the objective function (delta score).
\item Apply the first move that improves the objective function.
\item Repeat until no further improving move is found.
\end{DoxyEnumerate}

Solves a QAP instance with steepest local search. 

The function follows these steps\+:
\begin{DoxyEnumerate}
\item Generate all possible 2-\/opt moves (swaps of two positions).
\item Evaluate each move by computing its change on the objective function (delta score).
\item Apply the move that improves the objective function the most.
\item Repeat until no further improving move is found.
\end{DoxyEnumerate}

This heuristic provides a quick improvement over an initial solution and guarantees the final solution terminating in the local optimum.
