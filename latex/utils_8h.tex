The results are written in a tab-\/separated format for easy parsing. Each record is stored in a single line with the following structure\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{Algorithm:\ <algorithmName>\ \ \ \ Instance:\ <instance>\ \ \ \ Initial\ Score:\ <initialScore>\ \ \ \ }
\DoxyCodeLine{Score:\ <score>\ \ \ \ Number\ of\ Evaluations:\ <numEvaluations>\ \ \ \ }
\DoxyCodeLine{Number\ of\ Performed\ Moves:\ <numPerformedMoves>\ \ \ \ }
\DoxyCodeLine{Number\ of\ Best\ Solution\ Updates:\ <numBestSolutionUpdates>\ \ \ \ }
\DoxyCodeLine{Solution:\ <solution\_0>\ <solution\_1>\ ...\ <solution\_n>\ \ \ \ }
\DoxyCodeLine{Optimal\ Score:\ <optScore>\ \ \ \ }
\DoxyCodeLine{Optimal\ Solution:\ <optSolution\_0>\ <optSolution\_1>\ ...\ <optSolution\_n>}

\end{DoxyCode}


The file is opened in append mode ({\ttfamily ios\+::app}), ensuring that results from multiple runs do not overwrite previous entries.
